# Implementations

## TypeScript

The reference implementation of AAuth in TypeScript:

- [hellocoop/AAuth](https://github.com/hellocoop/AAuth) — TypeScript implementation of the AAuth protocol family.

## Other Implementations

Christian Posta's [AAuth Full Demo](https://blog.christianposta.com/aauth-full-demo/) includes working implementations in:
- **Java**
- **Python**
- **Rust**

These are part of a demo environment using Keycloak as the IdP and [Agentgateway](https://github.com/agentgateway/agentgateway) for policy enforcement.

*Building an AAuth implementation? [Open an issue](https://github.com/hellocoop/aauth.ai/issues) to get listed here.*
