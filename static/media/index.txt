# AAuth Media / Brand Assets

Logo files are free to use for AAuth-related content.

The AA monogram is the single mark across every AAuth touchpoint. It comes in
two forms:

- **AA Coin** — the monogram on its own dark background. Use this on surfaces
  you don't control (OAuth consent screens, "connected apps" lists, avatars).
  It carries its own background, so it looks identical on any host UI.
- **AA Logo** — transparent background. Use only where you set the background
  and have checked contrast. The bright green scores just 1.84:1 on white, so
  the transparent form is not safe on light surfaces.

## AA Coin (monogram on dark background — the robust default)

- [SVG](https://www.aauth.dev/media/aa-coin.svg) — scalable, viewBox 512×512, rounded
- [SVG, square](https://www.aauth.dev/media/aa-coin-square.svg) — full-bleed; use where the OS applies its own mask (iOS/Android)

padding and a brighter leading A, so the two A's stay distinct at favicon size.
